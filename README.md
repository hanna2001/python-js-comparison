## Declaring Variables
### Python
```
    a = 10
```
### Javascript
```
    let a = 10
```

## Unpacking/ Destructing
### Python
```
    fruits=['afdv','bvfv','cda']
    a,b,c=fruits
    print(a,b,c)
    ------------------------------------------------------
    afdv bvfv cda
```
### Javascript
```
    [a, b] = [10, 20];
    [a, b, ...rest] = [10, 20, 30, 40, 50];
```

## Operations
### Python
```
    + 	Addition 	    
    - 	Subtraction 	
    * 	Multiplication 	
    / 	Division 	   
    % 	Modulus 	    
    ** 	Exponentiation 	
    // 	Floor division 	
```
### Javascript
```
    + 	Addition
    - 	Subtraction
    * 	Multiplication
    / 	Division
    % 	Modulus (Division Remainder)
    ** 	Exponentiation (ES2016)
    ++ 	Increment
    -- 	Decrement
```
